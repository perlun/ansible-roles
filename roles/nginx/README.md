nginx
=====

Installs the [Nginx](https://www.nginx.org/) web server.

Requirements
------------

None

Supported Platforms
-------------------

- Ubuntu 18.04
- Raspbian 10
- Debian 10

(Might work on other platforms, but untested.)

Role Variables
--------------

See [defaults/main.yml](defaults/main.yml).

Dependencies
------------

No hard dependencies.
